# Centru de cursuri pentru copii - Clubul Copiilor

## Descriere
Acest proiect implementeaza o aplicatie de tip consola pentru gestionarea unui centru de cursuri destinat copiilor (de tip "Clubul Copiilor").

Aplicatia permite administrarea persoanelor implicate (copii, parinti, instructori, administratori), a cursurilor si a grupelor, programarea sedintelor pe intervale orare, precum si gestionarea platilor (abonament lunar si plata per sedinta).

Scopul proiectului este modelarea corecta a relatiilor dintre entitati folosind concepte de programare orientata pe obiecte in C++ (mostenire, polimorfism, incapsulare), conform cerintelor proiectului.

## Functionalitati principale
- gestionarea persoanelor (copil, parinte, instructor, administrator)
- gestionarea cursurilor si a grupelor
- programarea sedintelor folosind intervale orare
- inscrierea copiilor la cursuri, cu verificarea conflictelor de program
- gestionarea facturilor si a platilor (abonament lunar / plata per sedinta)
- meniu interactiv pentru ilustrarea functionalitatilor
